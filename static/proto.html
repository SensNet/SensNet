Sensor -> Node protocol version 4.0

Definitions

  • major version: breaking change
  • minor version: non-breaking change
  • The TTL is to be decreased with each hop form sensor to sensor and limited to 8bit. If ttl == 0 is reached, the packet must be dropped to avoid endless circles.

Punctuation is only for clarity.

Proto

Offset Length in bytes Type Description
0 1 uint_8 Major version (here: 0x04)
1 1 uint_8 Minor version (here: 0x00)
2 2 uint_16 Sensor identifier
4 1 uint_8 Sensor class
5 1 uint_8 Sensor type
6 4 int_32 Latitude
10 4 int_32 Longitude
14 1 int_8 Temperature
15 1 uint_8 Battery
16 4 uint_32 Time (hhmmss)
20 4 uint_32 Date (ddMMyy)
24 1 uint_8 TTL
25 1 uint_8 RSSI (assumed >= 0)
26 1 uint_8 Length
27 length uint_8 Data
